# 🎨 iDibujo

Version 1.3

iDibujo is a lightweight and intuitive iOS drawing application built with SwiftUI. Designed for simplicity and creativity, the app provides a smooth canvas experience that allows users to freely sketch, experiment with colors, and save their artwork directly to their device.

Whether you want to quickly doodle, practice digital sketching, or capture creative ideas on the go, iDibujo offers a clean and focused drawing environment without distractions.

## ✨ Features
🖌 Canvas Drawing

- Interactive drawing canvas built with SwiftUI’s Canvas

- Smooth freehand drawing using gesture recognition

- Real-time stroke rendering

## 🎨 Color Picker

- Native color picker integration

- Dynamic color selection for brush strokes

- Live color updates while drawing

## 📏 Adjustable Brush Size

- Slider control for brush thickness

- Real-time brush width adjustment

- Fine-grained stroke customization

## 🧰 Drawing Utilities

- Clear canvas functionality

- Simple and minimal tool interface

- Responsive gesture-based drawing input

## 💾 Save to Photo Library

- Export drawings directly to the device’s Photo Library

- Preserve artwork as an image

- Seamless saving experience with native iOS permissions

## 🛠 Tech Stack

- Swift

- SwiftUI

- Canvas API

- Gesture Recognizers

- PhotoKit (Photo Library Integration)

- Unit Testing (XCTest)

The application follows clean SwiftUI design principles with a focus on performance, state management, and responsive UI updates.

🧪 Testing

- iDibujo includes unit tests to ensure:

- Stroke data integrity

- Brush configuration logic

- Canvas state handling

- Utility behavior validation

Testing helps maintain reliability and supports future scalability.##
